# ContactFirst
iOS 11, Swift 4: Exploring how to use contacts

Important! Added this entry (drop down list) into **info.plist**, else app would crash:

``` 
Key   :  Privacy - Contacts Usage Description     
Value :  $(PRODUCT_NAME) contact use 
```
